# Qudrat ka Nizam Calculator

Pakistan T20 World Cup 2026 qualification scenarios - Group 2.

Pakistan must beat Sri Lanka AND finish with a higher NRR than New Zealand (+1.390) to qualify.

## Tools

### Scenario Calculator
Open [pakscenariosogk.netlify.app](https://pakscenariosogk.netlify.app) in any browser. Pick a scenario:

- **PAK Bats First** - Enter Pakistan's score, see the maximum Sri Lanka can score
- **PAK Chases** - Enter Sri Lanka's score, see how many overs Pakistan must chase in

### Live Match Tracker
Open [livescenarios.netlify.app](https://livescenarios.netlify.app) during the PAK vs SL match. Auto-refreshes every 30 seconds with live scores and qualification equation.

Note: Uses [cricketdata.org](https://cricketdata.org) free tier API (100 calls/day). If the page is blank, the daily limit has been hit.
